Afriflow AI is a production-ready AI automation platform I designed, built, and commercially deployed from scratch under my company, Bohlale Automated Services. The system includes a full client-facing marketing website, a PayFast-integrated checkout flow, a post-payment confirmation page, and a private admin dashboard with client management, revenue tracking, and system alerts. The product is actively marketed and sold to small businesses across South Africa, with packages ranging from R3,000 to R20,000 per month per client. Core services delivered include GPT-4 powered WhatsApp auto-reply bots, Google Calendar booking automation, CRM lead pipelines, and triggered email sequences. Built using HTML, CSS, and vanilla JavaScript on the frontend with Python Flask and the WhatsApp Business API powering the backend automation. This is my second commercial product under Bohlale Automated Services, following EASIstocks — a stock market education and analysis platform. Live site: https://afriflow-ai.onrender.com
